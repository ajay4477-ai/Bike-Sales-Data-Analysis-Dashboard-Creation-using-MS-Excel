# Bike-Sales-Data-Analysis-Dashboard-Creation-using-MS-Excel
This project focuses on analyzing a bike sales dataset containing 15 attributes related to customer demographics, financial background, and purchasing behavior. The goal of this analysis
was to identify key factors influencing bike purchases and generate actionable business insights using Microsoft Excel.
Two additional derived columns Age Group and Income Group were created to perform better customer segmentation and deeper analysis.
<br>

The dataset includes the following columns :
1) ID
2) Marital Status
3) Gender
4) Income
5) Children
6) Education
7) Occupation
8) Home Owner
9) Cars
10) Commute Distance
11) Region
12) Age
13) Purchased Bike
14) Age Group (Created during analysis)
15) ome Group (Created during analysis)
<br>

Objectives :
1) Identify customer segments most likely to purchase bikes
2) Analyze the impact of income, age, and region on buying behavior
3) Create an interactive dashboard for better visualization and decision-making
<br>

Key Analysis & Insights :
1) Segmented customers using Age Group and Income Group
2) Analyzed purchase patterns based on Gender and Marital Status
3) Studied the relationship between Income and Bike Purchases
<br>


Project Outcome :
This project demonstrates the ability to transform raw data into meaningful insights using Excel. The analysis helps identify target customer segments and supports data-driven decision 
making for business growth.
